## 2. 提供サービス・デジタルツール一覧

### 2.1 みらい議会（Mirai Gikai）

| 項目 | 内容 |
|------|------|
| URL | https://gikai.team-mir.ai/ |
| 公開日 | 2025年10月16日 |
| GitHub | https://github.com/team-mirai/mirai-gikai |
| 種別 | 国会法案可視化プラットフォーム |

**機能**:
- 国会で審議中の法案を一覧表示（カテゴリ別: 防災、人事、暮らし、医療・福祉、デジタル化等）
- 法案ステータス表示（審議前、審議中、成立等）
- 2種類の表示モード切替: 「やさしく」（平易な表現）と「詳しく」（専門用語含む）
- AIアシスタント機能（チャットで法案について質問可能）
- ルビ機能（ワンクリックで全漢字にふりがな表示）
- 改正案のポイント、法改正が必要な理由、賛成/反対意見、影響を受ける人/団体の表示
- チームみらいの賛否スタンス表示

**技術スタック**:
- Next.js（フロントエンド）
- Vercel AI SDK（LLMワークフロー構築）
- Langfuse（LLMの実行ログ管理・トレース・コスト管理）
- OpenTelemetry（トレース送信）

### 2.2 声が届くマニフェスト

| 項目 | 内容 |
|------|------|
| URL | https://policy.team-mir.ai/ |
| 公開日 | 2026年1月27日 |
| GitHub | https://github.com/team-mirai/policy（アーカイブ済）、https://github.com/team-mirai/manifesto-body |
| 種別 | 対話型政策プラットフォーム |

**機能**:
- 11の政策領域を網羅（子育て、産業、科学技術、教育、エネルギー、社会保障、医療、福祉、暮らし・行政、政治改革、その他重点政策）
- AI質問機能: マニフェスト内容についてチャットで質問可能
- 提案モード: 市民が政策改善提案を投稿可能
- 提案が検討され、良い提案は実際にマニフェストに反映
- リアルタイム統計表示（質問数: 約38,758件、改善提案数: 約6,198件）

**特徴**:
- 従来の「党が決めた後に公約発表」ではなく、有権者と共に政策を育てる参加型モデル
- マニフェストのバージョン管理（v0.1 → v0.2 → v1.0）

| 観点 | チームみらい | PoliPoli | 従来型政党 |
|------|-------------|----------|-----------|
| 性質 | 政党が自ら開発・運営 | スタートアップが運営 | 外部委託 or なし |

| 法案理解支援 | みらい議会（AI解説付き） | なし | なし |
| ファクトチェック | AIファクトチェッカー（OSS） | なし | 人力 or なし |
| AI対話 | AIあんの（3Dアバター+RAG） | なし | なし |
| ゲーミフィケーション | アクションボード（ポイント・ランキング） | なし | なし |

| ブロードリスニング | Talk to the City ベースの広聴AI | なし | なし |

### チームみらいの独自性

1. **政党自体がソフトウェア開発組織を内包** - 永田町にエンジニアチームを常駐させる唯一の政党
2. **全ツールがOSS** - 他党でも利用可能なインフラを構築
3. **LLMネイティブ開発** - 95%以上のコードをLLM生成、Langfuseによるオブザーバビリティ
4. **市民参加型の政策形成プロセス** - マニフェストのバージョン管理とGitHubベースの改善サイクル
5. **台湾のデジタル民主主義モデル（vTaiwan等）の日本実装** - いどばたシステムとして実現

## 8. まとめ

チームみらいは、単なる政党ウェブサイトを超え、**政治参加のための包括的デジタルエコシステム**を構築している。主要サービスは以下の7つ:

1. **みらい議会** - 法案の可視化・AI解説
2. **声が届くマニフェスト** - 対話型政策プラットフォーム
3. **まる見え政治資金** - 政治資金の透明化
4. **AIあんの** - AI政策応答アバター
5. **アクションボード** - ゲーミフィケーション型参加促進
6. **広聴AI / いどばた** - ブロードリスニング・大規模熟議
7. **AIファクトチェッカー** - SNS偽情報検証

これらのツールは全てオープンソースとして公開されており、政治のデジタル化を推進するインフラとして、チームみらいのみならず他の政党や政治団体でも活用可能な設計となっている。特に「まる見え政治資金」は公明党や自民党議員からも高い評価を受けており、超党派での導入が進みつつある。

技術面では、Next.js + Supabase + Vercel を共通基盤とし、LLMOps（Langfuse）を活用した効率的な開発体制を確立。永田町エンジニアチーム（8名）がLLMを駆使した高速開発を行い、約45日で中規模アプリケーションをリリースする能力を持つ。
